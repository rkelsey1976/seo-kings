## Posting on Facebook

**Organic post:** Copy the “Post / primary text” into your post, add the image (screenshot from the HTML file), and paste the link (seo-kings.co.uk or web-design-packages).  
**Facebook Ad:** In Ads Manager use the same text as primary text, add headline and description if the format allows, set CTA button and destination URL.

**Audience (Ads):** Location Bath, Keynsham, Midsomer Norton, Radstock, BANES (or radius). Interests: small business, local business, plumber, electrician, builder. Age 25–65.
