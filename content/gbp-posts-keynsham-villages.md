## Rotation

Ten posts. Post weekly, one village at a time, then cycle. Roughly a quarter per full
rotation, which is a sensible cadence — the same village twice in a month reads as spam.

Order by value: **Hanham → Longwell Green → Kingswood → Warmley → Brislington → Saltford →
Bitton → Whitchurch → Oldland Common → Stockwood**

Bigger places first (real search volume), villages later (quick wins, lower volume).

## Images

One per post, 1200x900 or larger. Best options in rough order:

1. **A real local photo** — the railway path for Warmley, the retail park for Longwell
   Green, Hanham High Street. Highest engagement, and it proves you know the place.
2. **A screenshot of a finished site** with the village name overlaid.
3. **A map graphic** with the postcode marked.

Avoid generic laptop stock photos — they read as filler and get scrolled past.

## Notes

- **Front-load the village name.** Only the first ~80 characters show before truncation in
  some views, so "Website design for Bitton businesses…" beats "Looking for a website
  designer? We cover Bitton…"
- **One CTA per post.** Learn more, pointed at the matching area page.
- **Don't post the same village twice in a month.** Ten posts on a weekly cycle handles this.
- **These pair with the area pages** rewritten on 2026-07-21 — each post's angle matches the
  content on its landing page, so someone clicking through gets the same argument expanded
  rather than a generic page.
- **Offer posts:** the free check works as an Offer-type post if you want extra visibility.
  Rotate one in every few cycles rather than every week.
