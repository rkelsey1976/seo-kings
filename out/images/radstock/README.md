# Radstock area page images

Optional Radstock-specific images for the area page (following the Keynsham layout). Add files here and update `AreaDetail.jsx` to use them:

- **radstock-hero.jpg** — Hero section (e.g. St Nicholas Church, High Street, or Radstock Museum). Currently the page uses a Wikimedia image.
- **radstock-intro.png** — Intro section (e.g. Radstock Museum, Victoria Hall). Currently uses a Wikimedia image.

Section images (why, services, CTA, etc.) can continue to use shared images from `/images/keynsham/` until Radstock-specific versions are added.
